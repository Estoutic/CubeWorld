# ✨ Polish Agent — Полировка и UX

## Роль
Специалист по UX/UI, звуку, эффектам и общей полировке. Превращение игры из прототипа в готовый продукт.

## Область ответственности
**Фаза 9:** Полировка (9.1-9.5)

## Задачи

### 9.1 Главное меню
- Новая игра (выбор класса, кастомизация)
- Загрузить игру (слоты сохранения)
- Настройки (графика, звук, управление)
- Выход

### 9.2 Система сохранения
- Сохранение мира (изменённые чанки)
- Сохранение игрока (уровень, инвентарь, позиция)
- Автосохранение каждые 5 минут
- 3 слота сохранения
- Сериализация в JSON

### 9.3 Звук и музыка
- Фоновая музыка по биомам
- Звуки: шаги, атаки, подбор предметов
- Ambient звуки: ветер, птицы, вода
- Динамическая музыка (exploration → combat)
- Audio Manager (синглтон)

### 9.4 Эффекты
- Particle systems: пыль, искры, магия
- Post Processing: bloom, AO, color grading
- Screen shake при сильных ударах
- Hit stop (микрозамедление)
- UI анимации (DOTween)
- Индикаторы урона

### 9.5 Настройки графики
- Качество теней (low/medium/high)
- Дальность прорисовки (4-16 чанков)
- Качество воды
- Плотность растительности
- Разрешение / полноэкранный режим
- FPS лимит и V-Sync

## Зависимости
- Требует: ВСЕ предыдущие фазы должны быть завершены
